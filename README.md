# 💻 Desarrollo de Aplicaciones Web

Este repositorio contiene los proyectos y ejercicios desarrollados durante el ramo de **Programación Web** de la carrera Ingeniería en Informática. [cite_start]El enfoque principal es la creación de interfaces interactivas, dinámicas y responsivas. [cite: 31, 32, 132]

## 🚀 Objetivo del Proyecto
Construir soluciones web modernas que integren una experiencia de usuario fluida con una lógica de programación eficiente, aplicando estándares actuales del desarrollo Front-end.

## 🛠️ Tecnologías Aplicadas
* [cite_start]**HTML5 & CSS3:** Estructuración de contenido y diseño visual avanzado con técnicas de Responsive Design. 
* [cite_start]**JavaScript:** Implementación de interactividad, manipulación del DOM y lógica en el navegador. 
* [cite_start]**Diseño UI/UX:** Enfoque en la usabilidad y navegación intuitiva para el usuario final. [cite: 131, 132]

## 🧠 Competencias Técnicas
* [cite_start]**Maquetación Responsiva:** Capacidad para adaptar aplicaciones a diversos dispositivos y tamaños de pantalla. 
* [cite_start]**Gestión de Eventos:** Programación de acciones dinámicas basadas en la interacción del usuario. 
* [cite_start]**Consumo de Estándares Web:** Uso de buenas prácticas de codificación para asegurar la compatibilidad y el rendimiento. [cite: 132, 189]
