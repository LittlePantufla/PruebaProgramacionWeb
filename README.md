# 💻 Desarrollo de Aplicaciones Web

Este repositorio contiene los proyectos y ejercicios desarrollados durante el ramo de **Programación Web** de la carrera Ingeniería en Informática. El enfoque principal es la creación de interfaces interactivas, dinámicas y responsivas. 

## 🚀 Objetivo del Proyecto
Construir soluciones web modernas que integren una experiencia de usuario fluida con una lógica de programación eficiente, aplicando estándares actuales del desarrollo Front-end.

## 🛠️ Tecnologías Aplicadas
* **HTML5 & CSS3:** Estructuración de contenido y diseño visual avanzado con técnicas de Responsive Design. 
* **JavaScript:** Implementación de interactividad, manipulación del DOM y lógica en el navegador. 
* **Diseño UI/UX:** Enfoque en la usabilidad y navegación intuitiva para el usuario final. [cite: 131, 132]

## 🧠 Competencias Técnicas
* **Maquetación Responsiva:** Capacidad para adaptar aplicaciones a diversos dispositivos y tamaños de pantalla. 
* **Gestión de Eventos:** Programación de acciones dinámicas basadas en la interacción del usuario. 
* **Consumo de Estándares Web:** Uso de buenas prácticas de codificación para asegurar la compatibilidad y el rendimiento.
